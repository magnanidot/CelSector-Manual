# Playing this Manual
The intended way to play this manual is using the save file provided. While you can technically use any 100% save file, the save provided is edited to have all items owned while still having the collectibles be present on the map, otherwise none of the clothing and weapon boxes will be present.

For help with setup search the oficial Manual for Archipelago [Discord](https://discord.gg/ptmNTJkVZ) or [GitHub](https://github.com/ManualForArchipelago/Manual/tree/main) page.

# Deathlink
If you are playing with the Death Link option enabled you should send a death whenever your Corruption gauge (the big heart) gets completely filled up. When recieving a death link pause the game and click "Exit to main menu" then click "Start game", this will send you to the last save point you used, which is exactly what happens when you "die"

# What does randomization do?
The following items are shuffled into the pool and can be anywhere in the multiworld:
- Access to each of the levels
- Ability to use each of the weapons
- Character customization items
- Mid-level roadblock removal (optionally)
- Each of the 4 unlockable cheat options (optionally)
- Access to gallery mode (optionally)
The following locations are checks where random items can appear:
-  Weapon Unlocks
- Customization item unlocks (from 1 to 7 checks in each clothing box)
- Path unlocks (such as generators and keycards)
- Defeating certain challenging rooms (like enemy gauntlets or mini-bosses)
- Watching each enemy's "attack" animation until the end (optionally)
- Saving in each of the game's save points (optionally)

# How to win?
There are 2 options you can choose as your goal:
- Beat the final boss! 
This option is meant to be shorter and has less required items, you win when you defeat the boss at the end of level C2. There is very strict weapon logic in place to not force a very difficult battle, as such it's definitely possible to beat it out of logic if you have access to level C2, enough ammo/DPS and patience.
- All levels complete! 
You goal when you check the "All levels complete!" location after getting every single Level Completion Token, if this is not your goal the level completion tokens and their associated checks have no use.
